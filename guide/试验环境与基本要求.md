## 2. Conda 虚拟环境（强制）

- **环境名称**：`agentsecurity`
- **Python 版本**：3.11.x（创建时指定 `python=3.11`）
- **环境路径（本机）**：`/Users/myw/anaconda3/envs/agentsecurity`

### 激活方式

```bash
conda activate agentsecurity
```

### 验证

```bash
python --version
pip --version
```

## 3. 依赖管理约定

- 新增 Python 依赖时，优先在本环境中用 `pip install` 或 `conda install` 安装，并**同步**将固定版本写入 **`repo/` 根目录**的 `requirements.txt`（若尚未创建，可在首次批量安装依赖时建立）；SC1 相关依赖见同目录下的 `requirements-sc1.txt`。
- 实验脚本、Notebook、与论文相关的数据处理代码，**默认解释器**为上述环境中的 `python`。

## 4. 科研与实验的基本要求

1. **可复现**：关键实验记录命令、随机种子（若使用）、数据与模型版本或路径说明（可在单独实验笔记或 `script` 目录中维护，本文件只约定环境层要求）。
2. **环境一致**：对外分享的步骤或 CI 中，应明确 `conda activate agentsecurity` 或等价方式（如 `conda run -n agentsecurity ...`）。
3. **安全与合规**：涉及调用外部 API、爬取数据或使用他人模型时，遵守平台条款与机构规定；敏感凭据不得写入仓库。
